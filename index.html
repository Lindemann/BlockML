BlockML

BlockML is a free and open source markup language for scientific documents. It is also pretty well suited for documentations, blog articles and other complex stuff.

The main goal of BlockML is to combine the most useful functions of LaTeX with the simplicity and advantages of lightweight markup languages like Markdown.

The BlockML OSX App turns minimally marked-up plain text into well formatted HTML documents, which are perfectly readable in the browser and on mobile devices. It comes with a nice default CSS theme, but basically, you can style your output HTML however you want.

The BlockML OSX App is basically a compiler for the BlockML markup language. To create plain text files you will need a separate text editor like TextMate.

Features And Syntax

BlockML provides a couple of features wich are especially needed for academic documents, like sections, tables of contents, bibliographic references or captions. BlockML also provides features wich are common for lightweight markup languages, like links, headings, images and so on.

For detailed information about the available markup elemensts and the syntax look into the documentation.

The syntax of BlockML is slightly S-Expression inspired and designed to be consistent and memorable. A BlockML markup element consists of a selector, who defines the type of the element and one or more blocks, which embed the text you want to mark up: selector[block]

fp[

title[Example Document Title]

h4[Author Name]

]


toc[Table of Contents]


sec[First Section][

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

sec[First Subsection][

math[$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}$$]

]/* First Subsection */

]/* First Section */

Get Started

1

Download the BlockML OSX App. In the download folder you will also find a TextMate Bundle file and two TextMate color scheme files

DOWNLOAD

2

Download TextMate 2 for free. The BlockML OSX App is basically a compiler for the BlockML markup language. To create plain text files you will need a separate text editor. You can use whatever text editor you like, but it is highly recommended to use TextMate in combination with the BlockML tmBundle and the tmThemes, to get syntax highlighting.

TEXTMATE

3

Install the tmBundle and tmThemes files on TextMate. Then set BlockML as language and choose between the "BlockML Day" or "BlockML Night" color scheme. For a detailed instruction read "Setup TextMate" in the documentation.

DOCUMENTATION

4

Copy and paste that example text into a .txt or .blockML file. Drag and drop this file onto the BlockML OSX App. Rainbows!
